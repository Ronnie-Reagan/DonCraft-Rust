What This Server Is

This is not a war server. It is not a true-roleplay server. It is not a rules-lawyer playground.

It is a community-focused vanilla Rust server where players build, cooperate, and create a shared world with minimal hostility.

The goal is a semi-persistent Rust world where social order exists because players value it, not because plugins force it.

Design Philosophy

  • Freedom without anarchy
  • Consequences without micromanagement
  • Community over competition

Server Features

  • No base decay
  • No raiding
  • Vanilla+ gameplay (deciding on long-term plugins carefully)
  • Property defense allowed
  • Toxicity and griefing removed early
  • Chat command for pasting images on signs
  • Features may expand based on community feedback
